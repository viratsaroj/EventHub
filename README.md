# EventHub
EventHub is a responsive college event portal built using HTML, CSS and JavaScript. It allows users to view and filter events dynamically, providing a clean and user-friendly interface.

The project displays various college activities such as workshops, seminars, and competitions.  
It includes events inspired by **Zealicon**, the college fest.

## Features

- Responsive layout using CSS
- Event cards interface
- Dynamic event filtering using JavaScript
- Basic login form UI
- Sticky navigation bar
- Social media integration
- Header and footer structure


## Technologies Used

- HTML
- CSS
- Bootstrap
- JavaScript (Fundamentals)

## Project Purpose

This project was created as a beginner-level web development practice project.

The main objective was to understand:
- Webpage structuring with HTML
- Styling with CSS
- Basic interactivity using JavaScript


## Functionality

- Displays multiple college events
- Allows filtering of events by category
- Fully responsive layout
